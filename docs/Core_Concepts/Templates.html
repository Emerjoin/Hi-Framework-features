    Remebering an Important rule about creating the html file of the template.

    The jquery script should always be included before the hi-framework script and they should both be placed as childs of head element.

    About Frontiers Handlers

    The best way to catch exceptions globally is to set a message on the promise to be exhibited by the global handler. The global handler receives the promise as first argument to be

     Frontier1.getFullName("Jon", "Snow")
     .try(function (data) {

            console.log(data);

      }).errorMessage = "The method error message";

    The global handler would display an error message this way:

    $frontiers:{
	catch : function(call,error){

		alert(call.errorMessage); //here you decide what to with the message

	}
}

    Changing your template

    To apply a diferent Template, inject the fronEnd and use the methdod:

       frontEnd.setTemplate("second") //receives a String with the name of the other template

    To get the current template use:

       frontEnd.getTemplate() //returns a String with the current template name
